Event Overview

**CODE MEET 2025** is an interactive seminar on **Agentic AI**, aimed at students, aspiring developers, and AI enthusiasts. The event focuses on practical training in **Git, GitHub, VS Code, CLI, and GitHub Copilot**, providing participants with hands-on experience in modern development workflows and collaborative coding practices.

Event Details

* **Venue:** CASPAM Lab, BZU Multan
* **Date & Time:** 11th December 2025 (Thursday) | 11:00 AM – 01:00 PM
* **Mode:** In-person

 Key Highlights

* Hands-on demonstrations of **Git and GitHub workflows**
* Usage of **VS Code** for efficient coding
* Introduction to **GitHub Copilot** for AI-assisted coding
* Collaborative project exercises and live Q&A sessions
* Guidance from experts in AI and software development
