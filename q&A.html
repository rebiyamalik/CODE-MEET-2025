Q & A

  1. What is Git?

    Git is a version control system that tracks changes in files and code.

  2. What is GitHub?

    GitHub is a cloud platform that hosts Git repositories and facilitates collaboration.

  3. What is VS Code?

    Visual Studio Code is a lightweight code editor with features like debugging, extensions, and Git integration.

  4. What is GitHub Copilot?

    GitHub Copilot is an AI-powered code assistant integrated with VS Code.

  5. How do I clone a repository?

    Use git clone repository-url to copy a repository to your local machine.

  6. How to initialize a Git repository?

    Use git init inside your project folder.

  7. How to create a new branch?

    Use git branch branch-name to create a new branch.

  8. How to switch branches?

    Use git checkout branch-name to switch to another branch.

  9. How to stage and commit changes?

    Use git add . to stage and git commit -m "message" to commit.

  10. How to push changes to GitHub?

    Use git push origin branch-name to push local commits to GitHub.

  11. How to view commit history?

    Use git log to see the history of commits.

  12. How to resolve merge conflicts?

    Edit conflicting files, stage them, and commit to resolve conflicts.

  13. How to configure Git username and email?

    Use git config --global user.name "Your Name" and git config --global user.email "you@example.com".

  14. How does GitHub Copilot help?

    It suggests code completions and generates snippets based on context.

  15. Can I work offline with Git?

    Yes, commits can be made locally and pushed when connected to the internet.
