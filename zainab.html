1. Introduction to Agentic AI

Agentic AI represents a new generation of AI systems capable of autonomous reasoning, planning, and tool usage. Unlike traditional chatbots, agentic AI systems can break down complex tasks into steps and execute them with minimal human input. Examples include GitHub Copilot Agent and Blackbox AI Agent.

2. Tools You Will Use

  • VS Code Insiders (required for Copilot Agents)
  • Git (version control)
  • Python (virtual environment + coding demo)
  • Node.js (optional for some agent extensions)
  • GitHub Copilot Agent
  • Blackbox AI Agent (browser-based)

3. Installing Requirements

  1. Install VS Code Insiders from Microsoft official website.
  2. Install Git.
  3. Install Python 3.10+.
  4. Install Node.js if needed.
  5. Enable GitHub Copilot extension + GitHub Copilot Agent.

4. Creating a Project in VS Code Insiders

Steps:

  1. Open VS Code Insiders
  2. Click File → New Folder
  3. Name it e.g., agentic-ai-demo
  4. Click Open Folder
  5. Right-click → New File → create main.py, etc.

5. Creating a Virtual Environment

python -m venv venv
venv\Scripts\activate  (Windows)
source venv/bin/activate  (Mac/Linux)

6. Installing basic libraries (Dependencies)

pip install requests openai fastapi uvicorn

7. Using GitHub Copilot Agent

Open Copilot Chat → Ask to generate project structure.

"Create a FastAPI API with a /predict endpoint."

8. Initializing Git & Creating GitHub Repository

git init
# Ask Copilot Agent: "Create a Python .gitignore file."
# Then push through VS Code or via Copilot Agent.

9. Introducing Blackbox AI Agent

Open blackbox.ai → Paste code → Ask for optimization, debugging, and explanations.

10. Final Workflow Summary

  • Explain Agentic AI
  • Install tools
  • Create project
  • Make virtual environment
  • Use agents for coding + structure
  • Create repo
  • Demo automation
