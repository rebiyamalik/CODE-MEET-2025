Overview

This page contains easy-to-read speaker notes for Git, GitHub and VS Code. Each section is written in plain language with practical commands and step-by-step actions.

Git

Definition

A version control system that tracks changes in code/files locally.

Key Points

  • Keeps history of every change
  • Allows reverting to previous versions
  • Supports multiple people working on the same project
  • Works locally without internet

Analogy

Git is like a notebook that tracks every edit you make to a story.

GitHub

Definition

A cloud platform to store Git repositories online and collaborate with others.

Key Points

  • Share code with others
  • Teams can collaborate remotely
  • Provides issue tracking, reviews, project management
  • Backup your code online

Analogy

GitHub is like a library where you store your notebook online so others can read or contribute.

Installations (Windows)

Install Git: Download from git-scm.com. After install, open Git Bash and run git --version.

Install VS Code: Download from code.visualstudio.com. Recommended extensions: GitLens, Prettier, GitHub Pull Requests.

Initial Setup (config)

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Create SSH Key (quick)

ssh-keygen -t ed25519 -C "you@example.com"
# then
cat ~/.ssh/id_ed25519.pub

Create, Init, Push (step-by-step)

mkdir myproject
cd myproject
code .
git init
git add .
git commit -m "first commit"
git remote add origin git@github.com:username/myproject.git
git branch -M main
git push -u origin main

Daily Workflow

git add .
git commit -m "describe your change"
git push

Branching & Merging

git checkout -b feature-xyz
# make changes
git add .
git commit -m "feat: add xyz"
git checkout main
git merge feature-xyz
git push
git branch -d feature-xyz

Delete SSH Key (GitHub)

Go to GitHub → Settings → SSH & GPG keys → Remove your key. To delete locally, remove the files from C:\Users\YourName\.ssh\.

VS Code

Definition

Visual Studio Code is a free, lightweight code editor with powerful features for coding, debugging, and version control integration.

Key Features

  • Syntax highlighting and IntelliSense
  • Integrated terminal
  • Debugging support
  • Extensions for Git, GitHub, and AI-powered tools like GitHub Copilot
  • Customizable themes and keyboard shortcuts

Useful Extensions

  • GitLens – enhances Git capabilities inside VS Code
  • Prettier – code formatter for consistent style
  • GitHub Pull Requests – manage PRs directly from VS Code
  • GitHub Copilot – AI-powered coding suggestions

Daily Workflow

# Open your project
code .

# Open terminal inside VS Code
# Stage your changes
git add .

# Commit your changes
git commit -m "Describe your changes"

# Push changes to GitHub
git push

Tips

  • Use the integrated terminal instead of switching to Git Bash
  • Use shortcuts like Ctrl+` for terminal, Ctrl+Shift+P for commands
  • Customize your editor with extensions for your workflow
